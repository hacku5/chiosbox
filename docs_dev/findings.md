# ChiosBox Araştırma Bulguları

## Proje Detayları
- **Proje Adı:** ChiosBox Dijital Platform
- **Tanım:** Türkiye'den AB'ye e-ticaret alışverişi yapanlar için paket konsolidasyon yönetim sistemi
- **Tasarım Dili:** Aegean Light (Ferah, Premium, Düşük Bilişsel Yük)

## Renk Paleti (Aegean Light)
| Renk | HEX | Kullanım |
|------|-----|----------|
| Mastic White | `#F9F9F7` | Arka plan |
| Chios Purple | `#5D3FD3` | Vurgu, butonlar |
| Deep Sea Teal | `#004953` | Metinler |
| Sunset Gold | `#FFCF7E` | Uyarı/dikkat |

## Tech Stack
- **Frontend:** Next.js (React) - PWA desteği
- **State Management:** Zustand + React Query
- **Styling:** Tailwind CSS + Radix UI
- **Animation:** Framer Motion + Lottie-React
- **Backend:** Node.js (Express veya NestJS)
- **Database:** PostgreSQL + Prisma ORM
- **Cron:** Node-Cron (gecikme ücretleri)
